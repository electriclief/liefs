Docs - Liefs Basic Concepts

What is a lief Item?
An Item is a window on your webpage. Items always have a "start" attribute.
This attribute is either fixed, (Constant number of pixels ie: "120px")
or Variable. (Percentage of remaining Container ie"100%")

What is a lief Container?
A Container is a row/column of Items. Containers can have 0 to any number of fixed Items in it.
Containers, by default are horizontal. To sepcify a Vertical Container, a vertical attribute must be assigned to non null.
The Variable Items must add up to 100%. (one "100%", or two ["20%","80%"] or any combo so long as they add to 100%)

Simple Example:
We want to make our own web site with a menu, (Shown In Red) on the left, with a constant width of 200px,
and a main body secion to show our content (Shown In Blue) on the right.

So, we need Two Items, and One Container:
Lets call the First Item "Tree" and the second "Main"
Lets call the container "_TreeMain" - using an underscore to remind us it is a container, not an item.
We would simply code:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">

    <!-- You must correctly link to this js file!!!! -->
    <script src="/js/basicNoexport.js"></script>

    <title>Too Easy To Be True</title>
</head>

<!--you must put onload="start" here!!! or it will do nothing-->
<body onload="start()">

<!-- created a container here -->
<liefs id="_TreeMain" contains="Tree|Main" ></liefs>

<!-- created two Items here -->
<div id="Tree" start="200px" style="background-color:red"></div>
<div id="Main" start="100%" style="background-color:blue"></div>

</body>
</html>
Now all we have to do is fill our div tags with our content, and we can forget about the layout. Its all handled automatically.
