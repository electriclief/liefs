Liefs Layout Manager - Introduction

Liefs is a HTML Dynamic Layout Manager. Its purpose is to allow web designers (or the average joe) the ability to easly manage the layout of their webpage.

Other layout managers do exist, but they didn't serve my needs.  This one does.  Lets take a quick look at an example.  Lets say, this page.  This page consists of:

  1. Horizontally, There is a Tree window on the left, 250px in width
  2. Horizontally, There is a Main window on the right, (where you are reading now) using the rest of the screen (100% of remaining)
  3. Vertically, There is Header window on top which is 64px in height.

This is represented in simple HTML like this. Please (Ctrl-u) See the source of this page to see how easy it is.

<liefs id="_master" contains="Header|_both" vertical="true"></liefs>
<liefs id="_both" contains="Tree|Main" start="100%"></liefs>

<div id="Header" start="64px"></div>

<div id="Tree" start="200px"></div>

<div id="Main" start="100%"></div>
Understanding the above code:
We have 3 Items Here: Header (Title Window), Tree (Menu Tree window), Main (Main Content window)
Items are always identified by their "start" attribute.
Header is 64px in Height
Tree is 250px in Length
Main is 100% of remaining Length

We have a horizontal Container "_both" (Use Underscore for containers -> just my habbit)
_both consists of the Tree Item(250px) , Followed by the Main Item (100%) of remaining. It is "Horizontal" because that is the default.

We have a vertical Container "_master"
_master consists of the Header Item (64px), Followed by the _both item (100%) of remaining. It is "Vertical" because vertical attribute was set to *anything*
